MEANINGS

  • ^ : Control key
  • + : Shift key
  • & : press first key then press second key
    exam) a & b : press and hold A, then press B

KEY MAPS

mode type hot key description example
- mode change Capslock swap INSERT/COMMAND mode Capslock
- mode change +Capslock upper case input mode +Capslock
- - Capslock & Space swap English/Korean mode Capslock & Space
- - Capslock & F4 Exit program Capslock & F4
- - Capslock & F5 Reload program Capslock & F5
COMMAND mode change (+) I insert I : insert
+I : insert at the beginning of the line
COMMAND mode change (+) O append a new line O : append a new line below the current line
+O : append a new line above the current line
COMMAND key input H Left H : Send Left Key
3H : Send Left Key 3 times
COMMAND key input J Down J : Send Down Key
27J : Send Down Key 27 times
COMMAND key input K Up K : Send Up Key
2K : Send Up Key 2 times
COMMAND key input L Right L : Send Right Key
2L : Send Right Key 2 times
COMMAND key input 0 Home 0
COMMAND key input +4 End : $ +4
COMMAND key input ^B Page Up ^B : Page Up
300^B : Send Page Up key 300 times
COMMAND key input ^F Page Down ^F : Page Down
4^F : Send Page Down key 4 times
COMMAND key input (+) X Delete X : do nothing. Just press X to delete is dangerous.
1X : Send Delete Key
10X : Send Delete Key 10 times
+X : Send BackSpace
7+X : Send BackSpace 7 times
COMMAND mouse input (^ +) Q Left Button of Mouse Q : Click left
^Q : Click left hold
+Q : unhold
100Q : Click left 100 times
COMMAND mouse input (^ +) E Right Button of Mouse E : Click right
^E : Click right hold
+E : unhold
10E : Click right 10 times
COMMAND mouse input R Wheel up R : Wheel up
5R : Wheel up 5 times
COMMAND mouse input F Wheel down F : Wheel up
50F : Wheel down 50 times
COMMAND mouse move [ move mouse pointer 400 pixel
COMMAND mouse move ] move mouse pointer 10 pixel
COMMAND mouse move \ move mouse pointer 1 pixel
COMMAND mouse move ([ ] \) W move mouse pointer up W : move mouse pointer up (70 pixel)
3W : move m-pointer up 3times (70*3 pixel)
[ & W : move m-pointer up (400 pixel)
] & W : move m-pointer up (10 pixel)
\ & W : move m-pointer up (1 pixel)
30 \ & W : move m-pixel up (30 pixel)
COMMAND mouse move ([ ] \) A move mouse pointer left like W
COMMAND mouse move ([ ] \) S move mouse pointer Down like W
COMMAND mouse move ([ ] \) D move mouse pointer Right like W

COMMAND mouse move Space & E move mouse pointer to top right

COMMAND mouse move Space & A move mouse pointer to middle left

: FUNCTIONS

in command mode, type ":" and input function name

function name description example
min minimize active window :min<ENTER>
max maximize active window :max<ENTER>
w send ctrl + s :w<ENTER>
trans adjust invisibility of active window. (0 ~ 255) :trans<ENTER>
:trans 0<ENTER>
:trans 255<ENTER>
notrans reset invisibility of active window to 255 :notrans<ENTER>
kill kill the process of active window :kill<ENTER>
top always on top this active window. (toggle) :top<ENTER>
color get rgb value of point at mouse pointer and input to clipboard :color<ENTER>
:<ENTER>
